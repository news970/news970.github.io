1. App Description

PDF Tools is a utility application that provides 12 tools for working with PDF files, including merging, splitting, compressing, rotating, deleting pages, converting images to PDF, exporting PDF to images, locking/unlocking PDFs, reordering pages, adding watermarks, adding page numbers, and extracting pages.

All PDF processing is performed entirely on your device using Apple's built-in frameworks. No files are uploaded to any server.
